Git本地仓库操作

发表时间：2022-07-27 阅读： 评论：

Git Git

在我们进行Git配置之后，建立了仓库。IDEA左侧目录文件颜色分为三种：黑色，绿色，红色。不同颜色代表不同状态

  • 黑色：代表该文件已经提交给你Git仓库。
  • 绿色：代表该文件已经添加到了暂存区
  • 红色：代表该文件还没有添加到暂存区

加入暂存区

每当创建新文件时，IDEA会提示你是否添加到暂存区。

手动加入暂存区

右键—>Git—>add【可能IDEA版本不同有一点差异，但差异不大】

然后文件颜色会从红色变为绿色。

提交到本地仓库

方式一

右键—>Git—>Commit File…

方式二

提交

提交之后颜色从绿色变为黑色

如果你一旦进行了修改，文件就从黑色变为蓝色

查看日志

IDEA左下角
