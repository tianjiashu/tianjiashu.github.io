SpringBoot读取Yaml配置

发表时间：2022-08-06 阅读： 评论：

Spring Yaml

读取配置文件内容三种方式：

  • @Value
  • Environment
  • @ConfigurationProperties

@Value方式

需要配置SpEL表达式，${一级属性名.二级属性名}

​

  • 对象形式

    • ```java
      @Value(“${server.port}”)
      private int port;

  • 普通形式

    • ```yaml
      @Value(“${name}”)
      private String name;

SpringBoot会通过数据装配的方式把数据全部装到这一个对象里。

使用getProperty，方法获取属性值。
